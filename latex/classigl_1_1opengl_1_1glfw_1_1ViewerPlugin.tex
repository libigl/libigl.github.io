\doxysubsection{Detailed Description}
Abstract class for plugins All plugins MUST have this class as their parent and may implement any/all the callbacks marked {\ttfamily virtual} here. 

Return value of callbacks\+: returning true to any of the callbacks tells \doxylink{classigl_1_1opengl_1_1glfw_1_1Viewer}{Viewer} that the event has been handled and that it should not be passed to other plugins or to other internal functions of \doxylink{classigl_1_1opengl_1_1glfw_1_1Viewer}{Viewer}
