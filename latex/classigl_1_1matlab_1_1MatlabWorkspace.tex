\doxysubsection{Detailed Description}
Class which contains data of a matlab workspace which can be written to a .mat file and loaded from matlab. 

This depends on matlab at compile time (though it shouldn\textquotesingle{}t necessarily have to) but it does not depend on running the matlab engine at run-\/time.

\begin{DoxyRefDesc}{Bug}
\item[\mbox{\hyperlink{bug__bug000005}{Bug}}]Treats all matrices as doubles (this may actually be desired for some \"{}index\"{} matrices since matlab\textquotesingle{}s sparse command takes doubles rather than int class matrices). It is of course not desired when dealing with logicals or uint\textquotesingle{}s for images. \end{DoxyRefDesc}
