\subsubsection*{template$<$typename DerivedV, int DIM$>$\newline
class igl\+::\+AABB$<$ Derived\+V, DIM $>$}
Implementation of semi-\/general purpose axis-\/aligned bounding box hierarchy. 

The mesh (V,Ele) is stored and managed by the caller and each routine here simply takes it as references (it better not change between calls).

It\textquotesingle{}s a little annoying that the Dimension is a template parameter and not picked up at run time from V. This leads to duplicated code for 2d/3d (up to dim).
